# OCR 识别工具 - 本地运行配置指南

## 🔑 重要提示

本工具使用**豆包 AI API** 进行 OCR 识别，需要配置 API 密钥才能正常使用。

## 📋 配置步骤

### 步骤 1：获取豆包 API 密钥

1. 访问 [豆包开放平台](https://www.coze.cn/open/oauth/apps)
2. 登录您的账号
3. 创建应用或使用现有应用
4. 获取 **API Key**

### 步骤 2：配置环境变量

1. 在项目根目录下创建 `.env.local` 文件

2. 复制以下内容并填入您的 API 密钥：

```env
COZE_API_KEY=your-coze-api-key-here
COZE_API_BASE=https://api.coze.com
```

> 将 `your-coze-api-key-here` 替换为您在步骤 1 中获取的实际 API Key

### 步骤 3：重启开发服务器

配置完成后，需要重启服务器：

```bash
# 停止当前服务器（按 Ctrl+C）

# 重新启动
pnpm next dev --port 5000
```

## ⚠️ 常见问题

### 问题 1：OCR 识别超时

**原因**：API 响应太慢或网络问题

**解决方法**：
1. 检查网络连接
2. 确认 API 密钥是否正确
3. 尝试使用较小的图片（小于 2MB）

### 问题 2：提示"OCR 识别失败"

**原因**：API 密钥未配置或已过期

**解决方法**：
1. 检查 `.env.local` 文件是否存在
2. 确认 API 密钥是否正确
3. 检查 API 密钥是否过期

### 问题 3：识别速度慢

**原因**：图片太大或并发太多

**解决方法**：
1. 压缩图片（推荐小于 2MB）
2. 减少每次处理的图片数量
3. 检查网络速度

## 💡 使用建议

### 图片要求

- **格式**: JPG、PNG、WebP
- **大小**: 推荐小于 2MB，最大 10MB
- **质量**: 中等质量即可，无需超高清
- **内容**: 文字清晰，光线充足

### 批量处理

- 推荐一次处理 10-20 张图片
- 如果有 1000+ 张图片，建议分批处理
- 预计每张图片识别时间：5-30 秒

### 性能优化

1. **压缩图片**：使用工具压缩图片大小
2. **降低并发**：已设置为 2 张并发
3. **增加超时**：已设置为 60 秒
